vivaUiPagination
directive in module ngVivaUi.pagination

Description

UI Kit's pagination implementation.

Usage

as element:
<viva-ui-pagination
       current-page="{number}"
       total-items="{number}"
       [items-per-page]="{number}"
       [start-at]="{number}">
</viva-ui-pagination>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
currentPagenumber

Current page.

totalItemsnumber

Total number of pages.

[itemsPerPage]number

Number of items per page. Default is 10.

[startAt]number

Reference used to start the pagination count. Default is 1.
