vivaUiPagination
directive in module ng-viva-ui.pagination

Description

UI Kit's pagination implementation.

Dependencies

  • ngVivaUi
  • dirPagination

Usage

as element:
<viva-ui-pagination
       start-at="{number}"
       current="{number}"
       total="{number}">
</viva-ui-pagination>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
startAtnumber

Number used to start the pagination count.

currentnumber

Current page.

totalnumber

Total number of pages.
